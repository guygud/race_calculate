# 🏎️ Racing Game Progression Calculator

Одностраничный онлайн-калькулятор прогрессии игрока по гоночной F2P-игре. Симулирует 1000 виртуальных игроков разных типов (от казуала до кита), показывает скорость прокачки, пейволлы и помогает настраивать экономику.

## Возможности

- **Monte Carlo симуляция** — 1000 игроков с разным поведением
- **5 типов игроков**: Хардкор-гриндер, Активный F2P, Казуал, Дельфин, Кит
- **Настраиваемые параметры**: гонки, награды, апгрейды, квесты, Battle Pass, скины
- **Шаринг через URL** — все параметры в query string, можно отправить коллеге
- **Графики**: прогрессия, пейволлы, баланс валюты, распределение
- **Индикатор здоровья экономики** — 🟢🟡🔴 по ключевым метрикам
- **Preset-профили**: текущий баланс, агрессивная монетизация, щедрый F2P

## Использование

1. Откройте `index.html` в браузере или разверните на GitHub Pages
2. Настройте параметры в левой панели (все секции сворачиваемые)
3. Симуляция запускается автоматически при изменении (debounce 500ms)
4. Результаты отображаются справа: KPI, таблицы, графики
5. Нажмите «📋 Скопировать ссылку» чтобы поделиться текущей конфигурацией

## Деплой на GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите файлы `index.html` и `README.md`
3. Settings → Pages → Source: Deploy from a branch
4. Branch: `main`, folder: `/ (root)`
5. Сохраните — через минуту калькулятор будет доступен по `https://<username>.github.io/<repo>/`

## Технические детали

- **PRNG**: Mulberry32 с seed из URL (seed=0 — случайный)
- **Производительность**: симуляция батчами по 100 с `setTimeout(0)`
- **Язык**: русский
